Part 1: Template Bot

The template bot is split into 4 active function and 1 helper function. The template bot choose a spot to play by randomly choosing a row; checks if it can play there; than repones with its chosen row. Lets meet the functions now.

start();

The function start is called every time the code is loaded, which is once every match (1000 games). It doesn't return anything back to the control server.
